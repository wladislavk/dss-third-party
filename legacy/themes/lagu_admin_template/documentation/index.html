01. Lagu Admin template

Lagu Admin is a HTML/CSS admin backend template enriched with many third party components (mostly jQuery plugins):

  • jQuery JS framework
  • jQuery UI user interface library
  • Fancybox tool for displaying images, HTML content and multi-media
  • Data Tables dynamic tables
  • Elfinder file explorer
  • File uploader A file upload script with progress-bar and drag-and-drop
  • Fushion charts Flash charting component
  • Gritter Growl look-a-like notification bubble/popup plugin
  • Chosen Select box enhancement plugin
  • Tiny MCE Javascript HTML WYSIWYG editor
  • Visualize jQuery charts
  • JQuery Micro Accordion Plugin Accordion
  • jQuery Tools Tabs Tabs
  • jQuery Tools Tooltip Tooltips
  • Text Overflow jQuery Text overflow plugin
  • xBreadcrumbs jQuery Breadcrumb plugin
  • Auto Expanding Textarea jQuery TextAreaExpander plugin
  • Content Slider jQuery Iphone style contact list plugin
  • JS validation jQuery plugin: Validation
  • Timepicker timepicker addon to jQuery UI datepicker
  • Iphone-like switches iPhone Toggle Switches
  • Rating A Star Rating Plugin
  • jQuery ScrollTo Easy element scrolling using jQuery
  • HighlightFade jQuery Plugin highlightFade
  • Calendar jQuery Plugin FullCalendar
  • Head JS JavaScript loader

02. File structure

General structure for Lagu Admin template:

  • Lagu admin/
    • css/ CSS files
    • gallery/ Gallery images
    • images/ Images and Icons used in this template
    • js/ Javascript plugins & configuration files
    • lib/ Third party components
    • *.html files & favicon.ico

03. HTML Structure & Layout

Layout

This template allows for both a fixed and a fluid width layout. For fixed layout (width: 960px) add class fixed to <body>, for fluid layout (full width) add class fluid to <body>.

Sidebar

To make sidebar visible add class sidebar to <body>.

07. Head JS & Plugins

Head JS

This template uses Head JS Javascript loader to load scripts.

Extended Documentation

Plugins

This template uses a lot of third party plugins. Plugins are split into 2 folders:
  • /js/ folder contains small scripts (jQuery tools, xbreadcrumbs, textarea-expander etc.), jQuery framework, Head JS loader and configuration files (lagu.js, login.js).
  • /lib/ folder contains more complex plugins (TinyMCE, Elfinder, Datatables etc.).
Configuration for all the plugins is in /js/lagu.js.
